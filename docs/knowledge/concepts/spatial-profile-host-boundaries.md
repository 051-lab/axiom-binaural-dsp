# Spatial Profile Host Boundaries

Type: concept note
Source IDs:

- `spatial-sound-principles-and-applications-xie`
- `spatial-hearing-revised-edition`
- `principles-and-applications-of-spatial-hearing`

## Purpose

This note separates spatial-audio system design from the accepted Axiom Core
EEL script. It is meant to guide future profile and host-path questions, not to
justify a Core candidate.

## Axiom Test-Design Use

Spatial rendering concepts should be classified before they become DSP work:

- Core: preserve current speaker-neutral enhancement behavior unless a
  repeatable Core problem is proven;
- Host path: crossfeed, headphone output, route selection, device EQ, and
  endpoint-specific behavior;
- Profile: deliberate headphone, immersive, reference, or production-listening
  behavior that is useful but not speaker-neutral Core behavior;
- Labs: exploratory spatial rendering, virtual-source, HRTF, Ambisonics, or
  externalization experiments that need isolated fixtures before any candidate
  exists;
- Qualification: listening records that distinguish spatial impression from
  tonal balance, loudness, fatigue, and device routing.

## Axiom Questions

- Does the idea require device-specific assumptions such as headphone response,
  HRTF choice, or route-dependent crossfeed?
- Would the change still make sense on speakers, or is it a future headphone
  profile?
- Is the goal localization stability, externalization, width, timbre
  neutrality, or only a more impressive spatial impression?
- What host settings must be declared before a listening record is comparable?
- Can the question be answered with vocabulary and listening records before any
  EEL candidate is created?

## Boundary

Spatial-audio references can sharpen Axiom profile design and listening
language. They do not prove Axiom behavior, and they do not justify adding
HRTF, Ambisonics, WFS, headphone correction, or virtual-source rendering to the
accepted Core EEL path without a scoped Labs plan, measurement where relevant,
controlled host execution, and human listening acceptance.
