## Summary

This source is a spatial-hearing and sound-localization reference focused on how
listeners perceive auditory direction, distance, image position, and spatial
events. For Axiom, it is most useful as background for test design around
headphone image stability, width changes, localization blur, and the difference
between acoustic cues and perceived spatial events.

## Axiom-Relevant Concepts

- spatial image stability: useful when judging whether width or bass changes
  make centered material wander, smear, or feel detached from the mix;
- localization blur: useful language for separating small image softening from
  obvious placement errors;
- interaural cue handling: relevant to crossfeed boundaries, headphone checks,
  and decisions about what belongs in the host versus the Axiom script;
- multiple-source perception: relevant when dense mixes make bass, vocals,
  cymbals, and ambience compete for stable placement;
- room and headphone distinction: useful for keeping speaker-neutral Core
  behavior separate from optional headphone listening preferences.
