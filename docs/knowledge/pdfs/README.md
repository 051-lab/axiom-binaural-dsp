# Local Knowledge PDFs

This directory is the local PDF shelf for Axiom Knowledge. PDF files in this
directory are ignored by git.

Use normalized source-ID filenames, for example:

```text
spatial-sound-principles-and-applications-xie.pdf
```

Register local files in `../source-index.local.json` using paths relative to
`docs/knowledge/`, such as
`pdfs/spatial-sound-principles-and-applications-xie.pdf`.

Do not copy book text into repo notes. Use citation metadata, short original
summaries, concept tags, and Axiom test-design questions.
